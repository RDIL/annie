Annie

What is Annie?

Annie is a small, lightweight, and fast analytics program for website owners.

Features

Annie has all these features (and more!) that make it amazing:

  • Speed - Annie is very fast to respond to API calls
  • Security - We keep your stats hidden, you control who gets to see them
  • Portability - The Annie backend server can run pretty much anywhere, so self hosting is an option
